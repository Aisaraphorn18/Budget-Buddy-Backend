# Documentation Images

## API Screenshots

### Create User API Interface
This folder contains screenshots and documentation images for the BudgetBuddy API.

### How to capture screenshots:
1. Run `python manage.py runserver`
2. Open browser and go to `http://127.0.0.1:8000/api/accounts/CreateUser/`
3. Take screenshot showing:
   - API description
   - Request/Response format
   - Input form
   - POST button

### Expected images:
- `create-user-api.png` - Create User API interface screenshot
- `api-root.png` - API root page screenshot
- `browsable-api-features.png` - DRF Browsable API features

### Notes:
Screenshots should be clear and show the full Django REST Framework interface
including the navigation, form fields, and response format options.
